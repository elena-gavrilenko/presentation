Regular Expressions in JavaScript

Supercharge your string manipulation skills!

What is a Regular Expression?

A sequence of characters that form a search pattern, used for string matching and manipulation. 


            const pattern = /hello/;

Creating a RegEx


            // Literal notation
            const regex1 = /abc/;

            // Constructor
            const regex2 = new RegExp("abc");

RegEx Flags

  • g – global search
  • i – case-insensitive
  • m – multiline mode

            const regex = /hello/gi;

Common Patterns

  • . – Any character
  • \d – Digit
  • \w – Word character
  • \s – Whitespace
  • + – One or more
  • * – Zero or more
  • ? – Optional
  • ^ – Start of string
  • $ – End of string

Testing Patterns


            const str = "Hello world!";
            const pattern = /world/;
            console.log(pattern.test(str)); // true

Extracting Matches


            const str = "Email me at hello@example.com";
            const pattern = /\S+@\S+\.\S+/;
            console.log(str.match(pattern)); // ['hello@example.com']

Replacing Text


            const str = "I like cats";
            const newStr = str.replace(/cats/, "dogs");
            console.log(newStr); // 'I like dogs'

Advanced Example


            const url = "https://example.com/page/123";
            const pattern = /\/page\/(\d+)/;
            const match = url.match(pattern);
            console.log(match[1]); // 123

Real-World Use Cases

  • Form validation (email, phone, etc.)
  • Search & highlight keywords
  • Data cleanup
  • Parsing logs or URLs
